Recipedia v2.1140 - 03-2020 (current release)

- added cb 1.14+ support, please backup and update config.yml
- added new recipe types (blast, smoking, campfire, stonecutting)
- improved targeting system for /recipetarget
- rewritten dictionary generation algorithms, robusted and optimized
- new name search algorithm: matches all non consecutive characters, case and accent unsensitive
- added 'reload' subcomand for ops, to flush and reload all recipes and dictionaries
- history system with a 'back' command instead of 'home' when click outside
- added hints to inventory titles (items found, recipe type)
- many other fixes and optimizations

known bugs:

- items set in stonecutter inventory show invisible, furnace inventory used as a temporary workaround
- cannot disable green recipe book events, seems a client feature (does anyone know how to disable advancements?)
- complex recipes not covered, see https://minecraft.gamepedia.com/Recipe#crafting_special_.2A

Recipedia v1 - 03-2019

- project name and version updated from RecipeSearch v3 to Recipedia v1
- total rewrite to add cb 1.13+ support
- many bug fixes (and some new, please report it) and additions from previous plugin
- updated commands (dropped /rc command, new commands added, type '/recipedia' for more info)
- better item browsing session handling
- new hitbox based collision algorithms for blocks and entities, for more accurate targeting
- reworked help pages
- added Material internal names to dictionary
